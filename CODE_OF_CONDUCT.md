# Code of Conduct

This project is committed to providing a welcoming and inclusive environment for everyone.

## Our standards

- Be respectful and considerate.
- Assume good intent.
- Avoid harassment, discrimination, or personal attacks.
- Focus on constructive collaboration.

## Enforcement

Violations may result in removal from project spaces or other appropriate actions.

## Contact

Please contact the maintainers through GitHub if you need to report an issue.

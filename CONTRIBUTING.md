# Contributing to TalentMind AI

Thank you for considering a contribution to TalentMind AI.

## How to contribute

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Add or update tests where appropriate.
5. Open a pull request.

## Development Guidelines

- Follow the existing feature-first backend structure.
- Keep changes focused and well-documented.
- Do not change the public architecture without discussion.
- Prefer small, reviewable pull requests.

## Code Style

- Python code should be formatted with Black.
- Use type hints where practical.
- Keep functions small and composable.

## Testing

Run the backend tests before submitting:

```bash
cd backend
pytest -q
```

## Reporting Issues

Please use the issue templates in [.github/ISSUE_TEMPLATE](.github/ISSUE_TEMPLATE).
